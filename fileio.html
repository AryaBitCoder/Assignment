File I/O

Files of the past can be changed through shock, encouragement or pain.

But in Programming world, files can only be changed using a file descriptor.

Problems

1. Write a program to create a file and write data provided via console.

2. Write a program to read data from a given file. File name should be provided by the user.

3. Write a program to copy data from one file to another.

4. Write a program to count the number of characters in a given file.

5. Write a program to print the source code of the same program.

6. Write a program to count the number of lines in file.

7. Write a program to append data at the end of a given file. The file will have some data in it.new appended data should not affect the previously written data.

8. Write a program to read the file and store the lines in an array.

9. Write a program to count the total words in a given file.

10. Write a program to count the frequency of a given word.

11. Write a program to delete the given word in file.

12. Write a program to find unique characters in a given file.

13. Write a program which will read integers from a file. The integers will be comma separated in the file. Program will read all integers and find the maximum number.

a. Input file: 123, 12, 10, 1234, 45

b. Output: 123

14. Write a program to read the following values from the file:

a. Name

b. Roll Number

c. Class

d. Address

Input file will have multiple lines of data, each line will represent a student. Input file:

• Abc, 1, 1, Delhi

• Xyz, 2, 4, Faridabad

• Aryabit, 2, 1, Gurugram
